### ADR-0024 — MapStruct Mapper Convention

- Library: `org.mapstruct:mapstruct:1.6.3`.
- Every mapper: `@Mapper(componentModel = "spring", unmappedTargetPolicy = ReportingPolicy.ERROR)`.
- Naming: class name = `{Concept}Mapper`, methods = `toDto` /
  `toEntity` / `toDomain`. Class carries the subject, method carries
  the direction.
- Multi-source escape hatch: `expression = "java(...)"` calling a
  `default` method on the interface.
- Annotation processor order: Lombok before MapStruct.
- `lombok-mapstruct-binding` deferred (YAGNI).
- 6 alternatives rejected (manual, ModelMapper, ReportingPolicy.IGNORE,
  componentModel=default, Lombok @Builder, @AfterMapping, splitting per
  direction).
- References cross-link to ADR-0003 (hexagonal — corrected from
  ADR-0001 after user caught the wrong number), ADR-0013 (Lombok),
  ADR-0023.

## Concepts explained / learned

- **`Optional` vs pattern matching for invariant extraction.** Six
  concrete reasons to prefer pattern matching, including Brian Goetz's
  intent for Optional (library return types signaling legitimate
  absence, not null-check replacement), type narrowing,
  fail-fast text placement, distinguishability of invariant vs domain
  absence, sealed-type composition, JIT cost, stack traces.
- **Spring Security 7 idioms.** `DelegatingAuthenticationEntryPoint`
  has three constructors: deprecated `LinkedHashMap`, modern
  `(default, List<RequestMatcherEntry>)`, varargs, and a
  `.builder()`. The builder is the new idiom.
- **`MediaTypeRequestMatcher` setters.** `useEquals(true)` for strict
  matching; `ignoredMediaTypes(MediaType.ALL)` so `*/*` does not match.
- **Multiple `SecurityFilterChain` beans** ordered by `@Order`;
  `FilterChainProxy` picks the first whose `securityMatcher` matches.
- **`@Profile("dev")` + `@Import` in slice tests.** Spring applies the
  profile filter even on explicit `@Import`, so toggling
  `@ActiveProfiles("dev")` controls whether the bean is in context.
- **Springdoc 3.0.3 `api-docs.enabled` gates Swagger UI autoconfig.**
  Undocumented; surfaced empirically as a 404 during the walk-through.
- **`/v3/api-docs/swagger-config` is the Swagger UI bootstrap
  endpoint.** Without it accessible, UI initialization fails silently.
- **`@WebMvcTest(controllers = {})` does not mean "no controllers".**
  Empty default means "include all". Excluding all controllers needs a
  marker class or `excludeFilters`.
- **`WebMvcAutoConfiguration` registers static resource serving in the
  slice.** `/openapi.yaml` returns 200 from
  `target/classes/static/openapi.yaml` if `maven-resources-plugin` has
  run. The slice is not as minimal as one might assume.
